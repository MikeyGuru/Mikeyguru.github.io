Smart Shopper!

Smart Shopper is the final project I created attending The Iron Yard. I used the Appcelerator Titanium SDK and wrote this as an Android native. It's a shopping list app that allows you to scan barcodes to add and remove items from your list. Smart Shopper calls to an API to get all the information about the product so it's virtually type free. Below is a video link you can watch to see how it works.

I used Javascript, Appcelerator Titanium SDK, SQLite Database, Semantics3 API, Android SDK, and JSON to build this app.
